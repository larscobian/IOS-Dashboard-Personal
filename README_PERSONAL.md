# Dashboard Personal 2026

Dashboard personal para el seguimiento de metas y hábitos durante el año 2026.

## Metas 2026

1. **Leer 5 libros** - Desarrollo personal y conocimiento
2. **Ahorrar $2,000** - Seguridad financiera
3. **Invertir $1,000** - Crecimiento patrimonial
4. **Ejercicio 3x/semana** - Salud física (156 sesiones al año)
5. **Meal Prep Domingos** - Organización alimenticia (52 semanas)
6. **Journal Diario** - Reflexión y registro (365 días)
7. **Dormir antes 00:00** - Descanso y rutina saludable (365 días)

## Estructura del Proyecto

```
src/
├── layouts/
│   ├── dashboard/     # Dashboard principal con resumen
│   ├── metas/         # Seguimiento visual de metas
│   ├── journal/       # Registro diario personal
│   └── [futuras]/     # Espacio para nuevas funcionalidades
```

## Funcionalidades Actuales

- **Dashboard Principal**: Vista general del progreso anual
- **Metas 2026**: Seguimiento detallado de cada meta con barras de progreso
- **Journal**: Sistema de registro diario para reflexiones

## Funcionalidades Futuras (Espacios Reservados)

- **Calendario**: Vista de calendario anual con marcadores de eventos y metas
- **Ejercicio**: Tracker específico para rutinas y sesiones
- **Meal Prep**: Planificador de comidas semanales con recetas
- **Estadísticas**: Análisis y gráficos de progreso a lo largo del tiempo

## Instalación

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm start

# Compilar para producción
npm run build
```

## Tecnologías

- React 18.2
- Material-UI 5.9
- ApexCharts para gráficos
- React Router para navegación
- Vision UI Dashboard como base

## Notas

Este proyecto es una adaptación del template Vision UI Dashboard React para uso personal.
Todas las funcionalidades están diseñadas para el seguimiento personal de metas 2026.
